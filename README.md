# AgroTech-Cloud-Management
Sistema de gestión ganadera y alertas de rotación de potreros basado en AWS y Python para optimización agrícola en Colombia.
# AgroTech Cloud Management 🐮🇨🇴

## Overview
Este proyecto nace de una necesidad real: digitalizar y optimizar la gestión de una finca ganadera familiar en Colombia. Utiliza **Python** y servicios de **AWS** para modernizar procesos tradicionales de agricultura.

## Key Features
* **Rotación de Potreros (1-7):** Sistema lógico que monitorea el tiempo de pastoreo para prevenir el sobrepastoreo y mejorar la salud del suelo.
* **Registro Sanitario Digital:** Seguimiento de vacunación, edades y pesaje de ganado mediante una arquitectura de base de datos NoSQL.
* **Alertas Cloud:** Notificaciones automáticas (AWS SNS/Lambda) para tareas críticas de la finca.

## Tech Stack
* **Language:** Python 3.x
* **Cloud:** AWS (Lambda, DynamoDB, SNS)
* **Data:** JSON / NoSQL
